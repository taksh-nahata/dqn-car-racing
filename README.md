# DQN Car Racing

This repository contains custom, scratch-built implementations of Deep Q-Networks (DQN) applied to Gymnasium's `CarRacing-v3` environment. The project includes a baseline DQN model, an architecturally refined Dueling DQN model, and a unified benchmarking framework to train and evaluate both models under identical conditions.

## Repository Structure

*   `dqn-car-racing.py`: Baseline DQN training script.
*   `dqn-refined-car-racing.py`: Refined Dueling DQN training script.
*   `benchmark_framework.py`: Unified, one-click script that trains both models sequentially and plots comparison data.
*   `dqn_learning_curve.png`: Training reward plot for the baseline model.
*   `dqn_refined_learning_curve.png`: Training reward plot for the refined model.
*   `benchmark_results.png`: Side-by-side comparison plot of rewards and training loss generated by the benchmark framework.
*   `Deep Q-Networks (DQN).pptx`: Slide deck explaining DQN mechanics.

## Algorithm Specs (PyTorch)

*   **Baseline DQN:** Uses a standard CNN processing visual frames to output Q-values directly. Uses a target network update frequency of 5 episodes and a replay buffer size of 10,000.
*   **Refined Dueling DQN:** Splits the network head into separate state-value V(s) and action-advantage A(s, a) streams to stabilize state evaluation. Uses a target network update frequency of 2 episodes, a slower epsilon decay of 0.93, and an expanded replay buffer size of 15,000.
*   **Experience Replay:** Uses a rolling buffer to store past transitions and sample random batches, breaking frame-to-frame correlation.
*   **Target Network:** Employs a frozen clone network to compute stable temporal difference targets.

## How to Run

To run the training or benchmarking headlessly inside a container (such as an NRP Nautilus pod), initialize the virtual frame buffer (`xvfb`):

```bash
# Install required packages
pip install gymnasium[box2d] pygame torch matplotlib

# To run the unified benchmark framework (runs both models and outputs comparison plots) use:
xvfb-run -s "-screen 0 1024x768x24" python benchmark_framework.py

# To run individual training scripts use:
xvfb-run -s "-screen 0 1024x768x24" python dqn-car-racing.py
# Or:
xvfb-run -s "-screen 0 1024x768x24" python dqn-refined-car-racing.py
